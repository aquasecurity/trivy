# Unpacked Filesystem

Scan an unpacked container image filesystem.

In this case, Trivy works the same way when scanning containers

```bash
$ docker export $(docker create alpine:3.10.2) | tar -C /tmp/rootfs -xvf -
$ trivy rootfs /tmp/rootfs
```
